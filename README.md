# Cp YT Source to MD
> Yeah, I know it is an anwful name

Firefox extesions create only to copy a YouTube video's title and link to system's clipboard. It uses the following Markdown format:

```
[tittle](link)
```

I programmed it because I use this format on my personal Obsidian notes and it's very annoying to put the video source of my notes by hand.
It doesn't use a real extension ID yet, so it isn't on the Firefox extesion library. I.e. you need to add it manually to you Firefox session using the debug tab.
